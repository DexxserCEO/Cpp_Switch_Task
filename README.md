# Cpp_Switch_Task
Практическая работа по оператору switch в C++
Для чего используется оператор switch?
 Оператор switch используется для выбора одной из нескольких возможных ветвей выполнения программы в зависимости от значения целочисленной или символьной переменной.

Чем switch отличается от условного оператора if?


switch предназначен для сравнения переменной с фиксированным набором значений (проверка на равенство), тогда как if (if...else if...else) позволяет проверять сложные логические выражения и диапазоны.

Какие типы данных можно использовать в выражении switch? 
Выражение в switch должно иметь целый или символьный тип (например, int, char, enum).


Чи допустимо использование такого запису: case 'A' ... 'B': ?
 Надайте пояснення. В стандартном C++ такой синтаксис недопустим. В некоторых компиляторах (например, GCC/Clang) этот синтаксис является расширением, позволяющим задать диапазон, но он не является переносимым.

Для чего нужен блок default у switch?
 Блок default является необязательным и выполняется, если значение выражения switch не совпало ни с одним из значений, указанных в блоках case.



Як можна об’єднувати кілька case для виконання одного коду?
 Можно перечислить несколько меток case последовательно, без оператора break между ними. Если сработает любая из этих меток, будет выполнен код, следующий за последней меткой.


У яких випадках доцільніше використовувати switch замість if…else?
 switch доцільно использовать, когда необходимо проверить одно значение переменной на множество дискретных равных значений (например, выбор по номеру меню или коду). Код при этом становится более читабельным и структурированным.

Як обробити помилкові значення користувача у switch? 
Ошибочные значения обрабатываются с помощью блока default. Если ввод пользователя не соответствует ни одному из case, выполняется код в default.


Що відбудеться, якщо у конструкції switch не вказати жодного break?
 Если не указать break после блока case, произойдет "провал" (fall-through), и выполнение кода продолжится в следующий блок case (или default), независимо от его условия, пока не будет достигнут break или конец switch.


Чи може в одному case бути кілька операторів або блок коду? Як це оформлюється? Да, может. После метки case можно разместить любое количество операторов, и они будут выполнены последовательно
